# WebTest - Vue 3 项目

这是一个基于 Vue 3 和 Vite 构建的现代化 Web 应用项目。

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 下一代前端构建工具
- **JavaScript** - 使用 Composition API

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

项目将在 `http://localhost:5173` 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 项目结构

```
WebTest/
├── index.html          # HTML 入口文件
├── vite.config.js      # Vite 配置文件
├── package.json        # 项目配置和依赖
├── src/
│   ├── main.js        # 应用入口文件
│   ├── App.vue        # 根组件
│   └── style.css      # 全局样式
└── README.md          # 项目说明
```

## 开发说明

- 使用 Vue 3 的 Composition API
- 支持 `<script setup>` 语法
- 使用 Vite 进行快速开发和构建
